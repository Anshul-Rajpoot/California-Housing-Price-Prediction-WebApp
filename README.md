This project is an end-to-end **Machine Learning + Flask Web Application** that predicts house prices using the **California Housing Prices Dataset** from Kaggle.

Users can enter property details (rooms, median income, population, etc.) through a simple web interface, and the app returns the predicted house price in real-time.
