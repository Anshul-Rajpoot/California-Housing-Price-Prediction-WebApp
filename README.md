# üöÄ California Housing Price Prediction ‚Äì ML Web App

This project is an end-to-end **Machine Learning + Flask Web Application** that predicts house prices using the **California Housing Prices Dataset** from Kaggle.

Users can enter property details (rooms, median income, population, etc.) through a simple web interface, and the app returns the predicted house price in real-time.

---

## üìå Dataset Source

**California Housing Prices Dataset (Kaggle):**  
https://www.kaggle.com/datasets/camnugent/california-housing-prices

This dataset contains housing information such as:

- Longitude & Latitude  
- Housing Median Age  
- Total Rooms & Bedrooms  
- Population  
- Households  
- Median Income  
- Ocean Proximity *(categorical)*  

**Target variable ‚Üí Median House Value**

## üß† Project Workflow

### 1Ô∏è‚É£ Data Loading & Exploration
- Used `pandas` to read and analyze the dataset  
- Performed stratified train-test split based on income categories  

### 2Ô∏è‚É£ Preprocessing Pipeline
- Handle missing values  
- Scale numerical features  
- One-hot encode categorical features  
- Combine using `ColumnTransformer`  

### 3Ô∏è‚É£ Model Training
- Algorithm used: **RandomForestRegressor**  
- Saved trained artifacts:
  - `model.pkl`
  - `pipeline.pkl`

### 4Ô∏è‚É£ Model Evaluation
- Evaluated using **RMSE (Root Mean Squared Error)**  
- Tested the model on unseen data  

### 5Ô∏è‚É£ Flask Web App
- HTML form for user input  
- Flask loads `model.pkl` and `pipeline.pkl`  
- Transforms input ‚Üí predicts price ‚Üí displays output
